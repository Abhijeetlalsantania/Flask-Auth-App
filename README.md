# 🔐 FlaskAuthApp

A simple and secure Authentication System built using **Flask**, **SQLite**, and **Bootstrap**.

This project includes user registration, login, session management, email validation, and account deletion functionality.

---

## 🚀 Features

- ✅ User Registration
- ✅ Email Format Validation (Server-side)
- ✅ Unique Email Check
- ✅ Password Minimum Length Validation (6 characters)
- ✅ Secure Password Hashing using bcrypt
- ✅ Login Authentication
- ✅ Session Management
- ✅ Protected Dashboard Route
- ✅ Delete Account Feature
- ✅ Flash Messages for Errors & Success
- ✅ Bootstrap UI

---

## 🛠 Tech Stack

- Python
- Flask
- Flask-SQLAlchemy
- SQLite
- bcrypt
- Bootstrap 5
